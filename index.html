HTML Table Reference

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The Set of rows defining the column headers in a table.
<tbody> Table Body The set of rows holding table data.
<tr> Table Row Creates row container.
<td> Table Data The table row where you input data.
<tfoot> Table Foot The set of rows defining the footer in the table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element spans across.
rowspan Row Span Defines how many rows a td element spans across.
